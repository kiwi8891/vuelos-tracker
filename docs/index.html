Cómo añadir / modificar búsquedas ▾
Edita config.json en el repo, haz commit y push.
El script se ejecutará con la nueva configuración en el siguiente cron (09:00 CET)
o puedes lanzarlo manualmente: Actions → Run workflow.

Para vuelos directos: "max_stops": 0 en outbound o return.
Para añadir un viaje nuevo: copia el bloque y cámbialo.
